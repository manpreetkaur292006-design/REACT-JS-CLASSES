# REACT-JS-CLASSES

A dedicated repository for React JS **class-based components** illustrations, examples, and learnings. Building on my previous JavaScript illustrations repo, this focuses specifically on React class components, lifecycle methods, state management, and practical exercises.

## 🚀 Features
- Class component fundamentals (`constructor`, `render`, `this`)
- Lifecycle methods (`componentDidMount`, `componentDidUpdate`, `componentWillUnmount`)
- State and Props with `this.setState()`
- Event handling and conditional rendering
- Hands-on coding examples and challenges
- Cross-references to my JS illustrations repo

## 📁 Structure
```
REACT-JS-CLASSES/
├── /basics/              # Constructor, render, JSX basics
├── /lifecycle/           # Lifecycle method examples
├── /state-props/         # State management & props passing
├── /events-forms/        # Event handlers & controlled components
├── /examples/            # Complete class component projects
├── /challenges/          # Practice exercises
└── README.md
```

## 🛠️ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/manpreetkaur292006-design/REACT-JS-CLASSES.git
   cd REACT-JS-CLASSES
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm start
   ```

## 📚 Learning Path

### 1. Basics
```
- Understanding 'this' in class components
- JSX syntax in render() method
- Props passing between class components
```

### 2. State Management
```
- Initializing state in constructor
- Updating state with setState()
- State vs Props
```

### 3. Lifecycle Methods
```
- Mounting: constructor() → render() → componentDidMount()
- Updating: shouldComponentUpdate() → render() → componentDidUpdate()
- Unmounting: componentWillUnmount()
```
